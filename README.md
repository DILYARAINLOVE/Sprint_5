# Проект Sprint_5 - UI тестирование сервиса "Доска"

# Описание проекта
Проект содержит автоматизированные UI-тесты для учебного сервиса "Доска".
Тесты написаны с использованием Selenium WebDriver и Pytest.

# Технологии
- Python 3.8+
- Selenium WebDriver 4.0+
- Pytest
- Google Chrome

# Структура проекта
README.md # Документация проекта
requirements.txt # Зависимости проекта
.gitignore # Игнорируемые файлы
conftest.py # Фикстуры Pytest
locators.py # Локаторы элементов
data.py # Тестовые данные
tests/ # Тестовые сценарии
init.py # Инициализация пакета
test_registration.py # Тесты регистрации
test_login.py # Тесты авторизации
test_logout.py # Тесты выхода
test_create_advertisement.py # Тесты создания объявлений# Sprint_5
